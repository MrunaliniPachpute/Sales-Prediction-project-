# Advertising Sales Prediction

A machine learning project that predicts product sales based on advertising budgets spent on TV, Radio, and Newspaper using Linear Regression.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## Dataset
AdvertisingBudgetandSales.csv  
Features: TV, Radio, Newspaper  
Target: Sales

## How to Run
1. Clone the repository  
2. Install dependencies: pip install pandas numpy matplotlib scikit-learn
3. Run the script: python sales_prediction.py

## Model
- Linear Regression
- Evaluation: MAE, MSE, RMSE, RÂ² Score
